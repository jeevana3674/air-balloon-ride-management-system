# 🎈 Air Balloon Ride Management System

## 📌 Project Description
The Air Balloon Ride Management System is a web-based application designed to manage commercial balloon ride operations efficiently. The system allows users to view available rides and book tickets, while administrators can monitor ride status, pilot availability, maintenance details, and final booked tickets through an admin dashboard. The project demonstrates how database systems can be used to organize and manage ride operations effectively.

---

## 🚀 Features

### User Features
- View available air balloon rides
- Book tickets for selected rides
- Enter passenger details
- Generate ticket confirmation
- View booking information

### Admin Features
- Monitor ride status
- Check pilot availability
- View maintenance status
- Track final booked tickets
- Manage seat availability for rides
